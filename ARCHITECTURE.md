# ARCHITECTURE.md - Portfolio BI & Analytics

This project is a personal portfolio and blog built with Astro and Tailwind CSS. It follows an agentic structure for development support.

## Project Structure

- `src/`: Source code
  - `components/`: Reusable Astro components
  - `content/`: Markdown content (blog, projects, docs)
  - `data/`: Structured data (site, experience, skills, education)
  - `layouts/`: Page layouts
  - `pages/`: Site routes
  - `styles/`: Global CSS and Tailwind configuration
- `public/`: Static assets
- `.agent/`: Agentic configuration and skills

## Agents (.agent/agents/)

- `orchestrator`: Multi-agent coordination
- `frontend-specialist`: UI/UX and Astro/React development
- `project-planner`: Task breakdown and planning
- `antigravity`: Main assistant agent (Antigravity Kit)
- *...and other specialized agents.*

## Skills (.agent/skills/)

- `clean-code`: Pragmatic coding standards
- `frontend-design`: UI/UX principles
- `nextjs-react-expert`: Performance and React patterns
- `plan-writing`: Structured planning
- `intelligent-routing`: Auto-agent selection

## Scripts (.agent/scripts/)

- `auto_preview.py`: Dev server management
- `checklist.py`: Quality audit
- `verify_all.py`: Full system verification
